Attention si quand tu te connectes sur Wifi et que des agents que tu connais pas se lance c'est que les autres sont aussi sur le wifi
faut que tu coupes ton wifi

Solo
1. Lancer le fichier BE-ID 
connecté au Wifi 5670 (en haut à droite) si c'est jaune c'est connecté
(si ça ouvre une pop up, il faut appuyer sur connect to the platform)

2. il faut utiliser le point .bat 
normalement ça va te lancer tous les agents 
et tu attends 5 secondes et tu peux jouer 

Multi 
1. Lancer un nouveau fichier vide
et après pareil que pour solo il faut se connecter au wifi en 5670

2. il faut lancer dans un cmd en se trouvant dans ingescape
il faut lancer le server avec
./sandbox/Server/main.py Server Wi-Fi 5670
et pour chaque instance du jeu que tu veux lancer tu fait 
./sandbox/mapper/main.py mapper Wi-Fi 5670
Pareil que pour Solo tu as environ 5 secondes d'attente 

=> si tout va bien tu as plein de fenêtre qui s'ouvre et plein de nouveaux agents dans l'ingescape circle 

Pour jouer dans le jeu tu as les touches suivantes:
A - rotation à gauche
E - rotation à droite
Z - avancer
S - reculer
Q - Aller à gauche
D - aller à droite
P - affiche la minimap
K - tuer un personnage

Pour lancer un agent dans cmd
chemin/du/script.py nomVouluPourl'agent Wi-Fi 5670

Pour lancer le whiteboard:
chemin/du/script.exe --device Wi-Fi --port 5670
J'ai pas trouver comment changer le nom du whiteboard il faut leur demander c'est un peu chiant

Comment créer un agent
dans ingescape il faut appuyer sur le +
ensuite tu rajoutes tes in et out
tu appuis sur generate code 
Attention choisi le bon agent sinon ça supprime le code (= moins bien que genome)
et après ça te génère le dossier avec le main.py

Ce qui reste à faire:
3 Agents:
Map:
in => impulsion start
out => string map 
Lorsque qu'on reçoit une impulsion sur start on envoi le string de la map sur le out
la map => X est un mur et . est un vide
R correspond à un retour à la ligne
exemple: 
XXXXXXXXXXRX........XRX........XRX...XXX..XRX...XXX..XRX...XXX..XRX........XRX........XRX........XRXXXXXXXXXX
map de 10x10

EventKeyboard:
out => impulsion Z
=> impulsion S
=> impulsion Q
=> impulsion D
=> impulsion A
=> impulsion E
=> impulsion ClickGauche(faudra que le map dans l'engine mais je m'en occupe)
=> impulsion M (music off/on)
=> impulsion N (sound off/on)
Lorsqu'on appui sur une touche envoi une impulsion sur le out adéquat

Player:
in => impulsion S
   => impulsion A
   => impulsion Z
   => impulsion E
   => impulsion Q
   => impulsion D
   => impulsion kill (lorsque joueur est kill, respawn pos random dans map pas dans le mur)
   => string map 
   => double degat (vie - degat > 0 -> kill)
out => int posX
    => int posY
    => double angle
    => int vie (pour engine)

quand on string la map la met à jour, tu as déjà un string map par défaut comme dans engine
les impulsions mettent à jour l'angle, la position tu peux voir dans engine pour les calculs 
il faut gérer les collisions, fait dans engine (normalement tu peux t'inspirer)
lorsque la pos ou l'angle change il faut indiquer sur les ports posX posY et angle
accumule les dégat et quand on atteint un certain seuil genre 100 même comportement que kill
